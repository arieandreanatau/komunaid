# User Requirements

**Proyek:** KomunaID
**Versi:** 1.0
**Tanggal:** 22 Juni 2026
**Status:** Draft

---

## 1. Visitor (Pengunjung yang belum login)

| ID | Requirement | Deskripsi | Prioritas |
|----|------------|-----------|-----------|
| UR-V001 | Melihat Landing Page | Visitor dapat melihat halaman utama platform dengan informasi tentang KomunaID | Must Have |
| UR-V002 | Melihat Daftar Komunitas | Visitor dapat melihat daftar komunitas publik yang terdaftar di platform | Must Have |
| UR-V003 | Mencari Komunitas | Visitor dapat mencari komunitas berdasarkan nama, kategori, atau lokasi | Must Have |
| UR-V004 | Melihat Detail Komunitas | Visitor dapat melihat detail komunitas (deskripsi, anggota, event, galeri) | Must Have |
| UR-V005 | Melihat Event Komunitas | Visitor dapat melihat daftar event yang dibuat oleh komunitas | Must Have |
| UR-V006 | Mendaftar Akun | Visitor dapat mendaftar akun baru sebagai member | Must Have |
| UR-V007 | Login | Visitor yang sudah terdaftar dapat login ke platform | Must Have |
| UR-V008 | Melihat Statistik Platform | Visitor dapat melihat jumlah komunitas, member, dan event | Should Have |

| UR-M003 | Leave Komunitas | Member dapat keluar dari komunitas yang diikuti | Must Have |

| UR-M005 | Mendaftar Event | Member dapat mendaftar event yang dibuat komunitas | Must Have |
| UR-M006 | Melihat Histori Event | Member dapat melihat daftar event yang pernah diikuti | Must Have |
| UR-M007 | Berdonasi | Member dapat berdonasi ke komunitas (opsional) | Must Have |
| UR-M008 | Melihat Histori Donasi | Member dapat melihat riwayat donasi yang pernah dilakukan | Must Have |
| UR-M009 | Mengajukan Role | Member dapat mengajukan role sebagai Community Owner atau Brand Owner | Must Have |
| UR-M010 | Melihat Dashboard | Member dapat melihat dashboard ringkasan aktivitas | Must Have |
| UR-M011 | Melihat Detail Komunitas Lengkap | Member dapat melihat detail komunitas termasuk galeri, chat, dan anggota | Should Have |
| UR-M012 | Melihat Kalender Event | Member dapat melihat event dalam bentuk kalender | Should Have |
| UR-M013 | Memberikan Rating/Komentar | Member dapat memberikan rating atau komentar pada event | Could Have |
| UR-M014 | Mengatur Notifikasi | Member dapat mengatur preferensi notifikasi | Could Have |

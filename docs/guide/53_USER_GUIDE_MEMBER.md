# 53 — USER GUIDE: MEMBER

## 53.1 Register
1. Buka https://komunaidv2-komuna.vercel.app/register.
2. Isi **email atau username** (minimal salah satu), password, dan konfirmasi password.
3. Klik **Daftar**.
4. Anda akan diarahkan ke halaman onboarding. Pilih interest Anda.

## 53.2 Login
1. Buka https://komunaidv2-komuna.vercel.app/login.
2. Isi email atau username + password.
3. Klik **Masuk**.

## 53.3 Lupa password
1. Klik **Lupa password** di halaman login.
2. Isi email Anda.
3. Cek inbox untuk link reset.

## 53.4 Cari komunitas
1. Klik **Komunitas** di navbar.
2. Gunakan filter region, kategori, atau jumlah member.
3. Klik komunitas untuk melihat detail.

## 53.5 Gabung komunitas
1. Buka halaman detail komunitas.
2. Klik **Gabung**.
3. Anda akan menjadi anggota. Aturan: maksimal 3x keluar-masuk dalam 90 hari (P1 enforcement).

## 53.6 Cari event
1. Klik **Events** di navbar.
2. Filter berdasarkan tipe (gratis / berbayar / donasi / info saja) dan visibilitas.
3. Klik event untuk melihat detail.

## 53.7 Daftar event
1. Klik **Daftar** di halaman event.
2. Jika gratis → konfirmasi.
3. Jika berbayar → ikuti instruksi pembayaran.
4. Jika donasi → masukkan nominal.

## 53.8 Ajukan volunteer
1. Buka event yang membuka volunteer.
2. Klik **Daftar Volunteer**.
3. Tunggu konfirmasi dari owner event.

## 53.9 Dompet
- Lihat saldo di **Member → Wallet**.
- Top-up akan tersedia di Phase 3 (payment gateway).

## 53.10 Donasi ke komunitas
- Buka halaman komunitas → klik **Donasi**.

## 53.11 Lapor konten
- Klik tombol **Laporkan** pada setiap konten.

## 53.12 Unduh laporan sendiri
- Buka **Member → Reports** → klik **Download CSV**.

## 53.13 Hapus akun
- Buka **Member → Settings → Delete Account** → konfirmasi.
- Akun akan di-*soft delete* selama 30 hari, lalu dihapus permanen.

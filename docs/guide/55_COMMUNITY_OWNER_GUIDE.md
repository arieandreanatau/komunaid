# 55 — COMMUNITY OWNER GUIDE

## 55.1 Submit komunitas
1. Daftar di `/register`, lalu ajukan role `community_owner` lewat **Onboarding**.
2. Setelah disetujui superadmin, buka **Community Owner → Buat Komunitas**.
3. Isi profil komunitas. Status awal = `pending`.
4. Tunggu approval. Anda akan menerima notifikasi.

## 55.2 Aturan penting
- Komunitas ke-2 baru bisa dibuat setelah komunitas ke-1 disetujui.
- Mode keanggotaan: **open**, **closed**, atau **approval**.

## 55.3 Sub komunitas & regional
- Tambah sub-komunitas dari menu **Subgroups**.
- Tambah regional dari menu **Regions**.

## 55.4 Kelola member
- **Members** → lihat, ubah role internal, ban, atau restore.
- Banned member tidak bisa join lagi sampai di-restore.

## 55.5 Buat event
- **Events → Create**.
- Pilih tipe: free, paid, donation, info-only.
- Pilih visibility: public, member, invitation.
- Submit → tunggu approval jika paid.

## 55.6 Buka volunteer
- **Events → Volunteer** → buka lowongan.

## 55.7 Kolaborasi
- **Collaborations** → terima / tolak / kirim proposal.

## 55.8 Donasi
- Donasi dari member muncul di **Donations**.

## 55.9 Laporan
- **Reports** → unduh CSV member, event, keuangan.

## 55.10 Tips
- Isi profil komunitas selengkap mungkin.
- Gunakan announcement untuk info penting.
- Backup data event sebelum publish.
